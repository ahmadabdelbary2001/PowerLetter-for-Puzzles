# API Documentation

## Internal Packages
- `@powerletter/core`: Core domain logic and state management.
- `@powerletter/ui`: Atomic design component library.
- `@powerletter/config`: Shared monorepo configurations.
- `@powerletter/api-bindings`: Shared API types and adapters.

## Rust Engines
- `power-engine`: Base traits and plugin system.
- `power-wasm`: JS/Rust bridge exports.
- `power-game-*`: Individual game-specific logic crates.
